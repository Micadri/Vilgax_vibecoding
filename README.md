# 🐙 Project Vilgax

**Vilgax** est un moteur d'automatisation axé sur le **SEO Programmatique** et la génération de pipelines de contenus/leads. Il permet de structurer des bases de données massives pour créer des pages optimisées à la volée.

## 🛠️ Stack Technique
* **Core :** JavaScript (Node.js) / Python
* **Automatisation :** Intégrations API (WordPress REST API, OpenAI / LLMs locaux)
* **Database :** SQLite / JSON

## 🚀 Mindset & Développement (Vibe Coding)
Projet propulsé par l'intelligence artificielle en mode **Vibe Coding**.
* **Mon rôle :** J'ai agi en tant que chef d'orchestre technique. Le défi majeur était de lier la logique marketing (SEO, structures de mots-clés) à une exécution technique propre. J'ai guidé l'IA pour assembler les briques API et gérer la logique métier.
* **Ce que j'ai appris :** L'importance du prompt-engineering avancé pour le code, la gestion des edge-cases dans les réponses d'API, et l'intégration du SEO technique dans du code automatisé.
